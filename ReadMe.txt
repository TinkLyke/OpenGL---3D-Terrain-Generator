Instruction
---------------------------------
Keyboard functions
'Q','q' - quit

'R' 	- reset

'r'	- random map

'W','w'	- switch between frames

'F','f'	- change shading format

'T','t'	- switch to triangle_strip

'Y','y'	- switch to quad_strip

'L','l'	- switch light on/off

'C','c'	- switch to color mode

'U','u'	- move ligth0
